## 西田君
### 観光地
- 鏡山公園  
国史跡・鏡山城跡の麓に整備された公園で、30種余り約500本の桜が咲きほこる、東広島を代表する桜の名所の一つ。
- 三ッ城古墳  
西条盆地の中央、八幡山から北に伸びる丘陵の先端に三ッ城古墳は造られました。全長約92メートル、後円部の直径約62メートル、高さ約13メートル、前方部の幅約66メートルで、国史跡に指定されている県内最大の前方後円墳です。
